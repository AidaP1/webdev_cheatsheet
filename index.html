Cheatsheet: Semantic HTML

Tag Name Description
<header> Header Used to hold nav links and introductory content
<nav> Nav Used to hold navigational links, such as menus or a table of contents. Often inside <header>
<main> Main used to surround the dominant content in a webpage
<footer> Footer Sits at the bottom containing info such as contact details, terms of use, etc.
<article> Article Holds content that makes sense on it's own, like a blog post or block of self-contained info.
<section> Section Defines elements in a document, like a chapter or heading, anything that comes together under the same theme.
<aside> Aside Used to mark out additional information that can enhance another element, but the other element can still be understood without
<figure> Figure Used to encapsulate media such as an image, diagram, or code, that will be referenced in the main body of the doc
<figcaption> Figcaption Adds a caption to an element and should be placed inside <figure>, allowing them to be moved around together
<audio> Audio used to embed audio content, it contains an <source>
